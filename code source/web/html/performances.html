Taux d'erreur

Ce pourcentage a été obtenu à partir d'un test de reconnaissance sur 1200 images tout en comptant le nombre de fois où le résultat n'était pas le bon

Faux negatifs

Un faux négatif est lorsque l'algorithme ne considère pas le chiffre représenté sur l'image car le résultat de son interprétation est inférieur au seuil déterminé.

Faux positifs

Un faux positive est lorsque l'algorithme considère des lettres / caractère entrées par les images importées comme étant des chiffres.
